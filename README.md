## Prerequisites

Before you begin, ensure you have met the following requirements:

* [Git](https://github.com/yousaf4995/portfolio.git"Download Git") must be installed on your operating system.

## Installing vCard

To install **vCard**, follow these steps:

Linux and macOS:

```bash
sudo git clone https://github.com/yousaf4995/portfolio.git
```

Windows:

```bash
git clone https://github.com/yousaf4995/portfolio.git
```
